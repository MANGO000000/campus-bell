## 📌 프로젝트 소개

캠퍼스벨은 학생들이 놓치기 쉬운 **장학금**, **비교과 프로그램**, **대외활동** 정보를 한 곳에서 확인하고 관리할 수 있는 웹 플랫폼입니다.

### 🎯 주요 기능

- 🔔 **실시간 알림**: 마감 임박한 공지를 놓치지 않도록 D-day 알림
- 💰 **장학금 정보**: 교내/교외/국가/기업 장학금 통합 관리
- 📚 **비교과 프로그램**: 특강, 동아리, 봉사, 경진대회 정보
- 💼 **대외활동**: 공모전, 인턴, 교육 프로그램 정보
- ⭐ **북마크**: 관심 있는 공지 저장
- 📅 **캘린더**: 월별 일정 한눈에 보기
- 🔐 **회원 관리**: Firebase Authentication 기반 안전한 로그인

## 🛠️ 기술 스택

### Frontend
- **HTML5** - 웹 구조
- **CSS3** - 스타일링 (카드형 UI)
- **JavaScript (ES6+)** - 동적 기능 구현

### Backend
- **Firebase Authentication** - 사용자 인증
- **Cloud Firestore** - NoSQL 데이터베이스
- **Firebase Hosting** - 웹 호스팅

### Development Tools
- **Firebase CLI** - 배포 및 관리
- **Git/GitHub** - 버전 관리
